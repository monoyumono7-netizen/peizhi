# TestBuddy IDE 插件

TestBuddy IDE 插件提供 AI 测试设计能力，通过 Skill 技能包的形式集成到 IDE Agent 中。

## 功能介绍

- **拿到需求就能出用例**：粘贴 TAPD 需求/缺陷链接、PRD 或接口文档，AI 自动生成完整的测试用例
- **在脑图里直接做测试设计**：打开 TestBuddy 脑图画布，触发 AI 后结果实时写入画布，边设计边调整
- **代码改了快速找到关联用例**：记录用例与代码路径的映射，代码变更时可快速定位需要回归的测试范围
